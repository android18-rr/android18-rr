Memos for Github pages with Hugo

初期設定

Hugoのインストール

注意点はaptでインストールされるバージョンがとても古い。古いバージョンだとテーマが対応していない場合もあるので、最新バージョンのdebパッケージをHugoのGithubリポジトリからダウンロードして、dpkgでインストールする。

サイト生成

サイト生成は下記の手順。

  1. サイトのトップディレクトリ生成

    • hugoが自動でpostなどの各種ディレクトリを生成。

  2. サイトのテーマ選択

    • submoduleかcloneでtheme以下にthemeディレクトリを持ってくる。
    • config.tomlを設定するが、theme毎に異なるので、themeの説明を参照する。

  3. コンテンツ(ページ)の作成

    • Markdownで記事を書く。
    • themeによって注意事項がある。例えば、本例のgokarnaはtypeをpostにするとpostに勝手に入れてくれたりする。

  4. Live Serverで確認

    • ローカル(localhost:1313)で記事を確認。

  5. サイト生成

    • 生成するとpublicディレクトリにサイトが生成されている。

  6. Githubリポジトリとトップディレクトリを接続(初回のみ)。

    • public以下をプッシュするように、publicディレクトリとGithubのGithub Pagesリポジトリをリンクする。

  7. Githubにプッシュ。

# (1) Generate top directory
$ hugo new site <top_dir>

# (2) Import theme
$ cd <top_dir>
$ git init
$ git submodule add https://github.com/526avijitgupta/gokarna.git themes/gokarna
$ vi config.toml  # Edit configuration file as you need.

# (3) Generate and edit new contents
$ hugo new post/contents/hoge.md
$ vi hoge.md  # edit the content

# (4) Check the contents
$ hugo server -D  # generate site and run local server.
$ firefox http://localhost:1313

# (6) Generate site
$ hugo

# (5) add repository



# (7) Push the site
